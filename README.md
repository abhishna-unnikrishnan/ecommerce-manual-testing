E-Commerce Manual Testing Project

## Project Overview
This project focuses on manual testing of a demo e-commerce web application.
The goal is to validate core business functionalities such as user authentication,
product selection, cart management, and checkout flow.

## Application Under Test
- Type: Web Application
- Domain: E-Commerce
- URL: https://demo.opencart.com

## Scope of Testing
- Functional Testing
- UI Testing
- Regression Testing
- Smoke Testing

Test Artifacts
- Test Plan
- Test Scenarios
- Test Cases
- Bug Reports
- Test Summary Report

 Tools Used
- Test case documentation: Excel
- Bug tracking: JIRA (sample reports)
- Screenshot capturing for defects

Key Modules Tested
- Login & Registration
- Product Listing
- Add to Cart
- Checkout
- Payment
